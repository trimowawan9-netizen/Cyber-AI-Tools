1.2 Linux Command

ls, cd, grep, chmod, sudo. Wajib hafal 20 command dasar

20 Command Wajib Hafal

1. pwd - Liat posisi kamu sekarang
2. ls - Liat isi folder
3. cd - Pindah folder
4. mkdir - Bikin folder baru
5. touch - Bikin file kosong
6. rm - Hapus file/folder
7. cp - Copy file
8. mv - Pindah/rename file
9. cat - Baca isi file
10. grep - Cari kata di file
Latihan: Buka terminal, ketik pwd lalu ls -la. Itu command paling basic hacker
