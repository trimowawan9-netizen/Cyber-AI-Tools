1.3 Network Basics

IP Address, TCP/IP, DNS, Port. Bahasa komputer biar bisa ngobrol

1. IP Address = Alamat Rumah Komputer

Setiap device di internet punya alamat unik. Format: xxx.xxx

Contoh: 192.168.1.1 = Router rumah
8.8.8.8 = DNS Google
127.0.0.1 = Localhost = komputer kamu sendiri

2. Port = Pintu Rumah

1 IP punya 65535 port. Tiap port buat layanan beda

PortLayananCatatan Hacker
21FTPTransfer file - rawan
22SSHRemote server Linux
80HTTPWebsite biasa
443HTTPSWebsite aman/gembok

3. TCP/IP = Aturan Ngobrol

Protokol biar data gak nyasar. TCP = pasti nyampe, UDP = cepat tapi bisa hilang

4. DNS = Buku Telepon Internet

Ngubah nama jadi IP. google.com → 142.250.185.78

Command wajib:
ping google.com - Cek koneksi + liat IP
nslookup google.com - Liat IP dari nama domain
🔥 Latihan: Buka CMD/Terminal ketik ping 8.8.8.8 Enter. Kalo ada "Reply from" = internet kamu nyambung
