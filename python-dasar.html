1.4 Python Dasar

Syntax, variable, loop, function. Bahasa utama buat bikin tools hacker

1. Variable = Kotak Penyimpanan

Tempat nyimpen data. Python gak ribet, gak pake tipe data

nama = "Bro Hacker" umur = 20 aktif = True print(nama)

2. Loop = Kerjain Berulang

Buat bruteforce, scan port, spam request

for i in range(5): print("Coba login ke-", i)

3. If Else = Logika Keputusan

Buat bikin kondisi: kalo bener gini, kalo salah gitu

password = "admin123" if password == "admin123": print("Berhasil masuk!") else: print("Password salah")

4. Function = Kotak Ajaib

Kumpulin kode biar dipake berulang. Hacker wajib pake ini

def scan_port(ip, port): print(f"Scan {ip} port {port}") return "Open" hasil = scan_port("192.168.1.1", 80)
🔥 Kenapa Python Wajib?
1. 90% tools hacker pake Python: Nmap, Metasploit, Burp Suite
2. Syntax paling gampang dibanding C/C++
3. Library banyak: requests, socket, scapy
