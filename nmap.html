2.4 Tools Nmap

Scan port, OS detection, Script engine. "Google Maps" buat hacker

⚠️ ETIKA HACKER: Scan cuma di jaringan kamu sendiri atau izin tertulis. Scan IP orang = ilegal!

1. Nmap = Alat Scan Port

Tugas nmap: ngetok semua pintu rumah target, liat pintu mana yang kebuka + pintunya buat apa

nmap target.com
→ Scan 1000 port paling umum. Output: port 80 open, 443 open

nmap -sV target.com
→ Version scan. Ketauan Apache 2.4, OpenSSH 7.2

nmap -O target.com
→ OS detection. Ketauan target pake Linux/Windows

nmap -sC target.com
→ Script scan. Auto cek vulnerability basic

2. Baca Hasil Nmap Kayak Pro

PORT STATE SERVICE VERSION
22/tcp open ssh OpenSSH 7.2p2
80/tcp open http Apache 2.4.18
443/tcp open https Apache 2.4.18

open = Pintu kebuka
closed = Pintu ada tapi dikunci
filtered = Ada firewall, gak bisa dicek

3. Command Combo Wajib Hafal

nmap -sV -sC -O -p- target.com
Artinya: Scan semua port + versi + OS + script
Ini scan lengkap tapi lama + berisik, gampang ketauan IDS

4. Install Nmap

Linux: sudo apt install nmap
Windows: Download di nmap.org, namanya "Zenmap" ada GUI-nya
Termux: pkg install nmap
🔥 Latihan Aman 1 Menit:
1. Scan diri sendiri dulu: nmap localhost
2. Atau scan router rumah: nmap 192.168.1.1
3. Liat port 80, 443 kebuka gak. Kalo ada = web router jalan
JANGAN SCAN IP PUBLIK ORANG TANPA IZIN!
