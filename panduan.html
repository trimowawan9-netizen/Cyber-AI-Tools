📖 Panduan Lengkap

1. Link Phishing Checker

Paste link mencurigakan > Klik Analyze. Ciri bahaya: bit.ly, kata free/hadiah, minta login.

⚠️ Jangan klik link buru-buru "24 jam"!

2. Password Checker

Ketik password > Liat bar warna. Tips kuat: 12 karakter + A-Z + 0-9 + simbol.

3. Email Scam Detector

Paste isi email "Menang Hadiah" > Scan. Kalo ada kata OTP/Transfer = SCAM.

4. Mode Darurat

Tombol Google buat ganti password. Tombol Copy buat chat CS kalo akun dibobol.
