📖 Panduan Lengkap AI Cyber Security Tools

1. Link Phishing Checker

Fungsi: Cek link mencurigakan dari WA/Email/Telegram

Cara pakai: Paste link > Klik "Analyze Link Sekarang"

⚠️ Ciri link berbahaya: bit.ly, kata "free/hadiah", minta login, buru-buru "24 jam"

2. Password Strength Checker

Fungsi: Cek seberapa kuat password kamu

Cara pakai: Ketik password > Liat bar warna + teks kekuatan

Tips password kuat: Min 12 karakter + Huruf besar + Angka + Simbol !@#$%

3. Email Scam Detector

Fungsi: Deteksi email "Menang Hadiah" palsu

Cara pakai: Copy isi email > Paste > Klik "Scan Email Sekarang"

⚠️ Kalo ada kata "OTP/Transfer/Klaim sekarang" = 99% SCAM

4. Mode Darurat Akun

Buka Keamanan Google: Langsung ke myaccount.google.com/security buat ganti password + cek login mencurigakan

Copy Pesan WA: Template chat ke CS Google kalo akun dibobol
