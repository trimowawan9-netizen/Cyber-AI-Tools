3.5 Burp Suite Community

⚠️ ATURAN WAJIB: Burp Suite cuma boleh dipake buat test web milik sendiri, DVWA, PortSwigger Lab. Scan web orang tanpa izin = cybercrime. Burp ada mode passive, jadi aman buat pemula.

1. Apa itu Burp Suite?

Burp = "Proxy" buat hacker baik. Ibarat kaca pembesar + editor buat liat semua request yang keluar dari browser kamu ke web. Versi Community GRATIS udah cukup buat belajar.

2. 4 Fitur Utama Versi Gratis

1. Proxy
Nangkep request sebelum dikirim ke web. Kamu bisa liat, edit, terus forward. Guna: Paham gimana data login dikirim, cek ada CSRF token gak.
2. Repeater
Kirim request yang sama berkali-kali tapi ganti isi parameter. Guna: Test manual SQLi/XSS di lab DVWA. Gak ada auto-attack di versi gratis.
3. Decoder
Enkripsi/dekripsi Base64, URL, MD5, dll. Guna: Baca token JWT, decode cookie.
4. Scanner Passive
Versi gratis cuma scan pasif = gak nyerang. Cuma ngasih tau "nih kayaknya ada potensi XSS di sini". Akurat 100% aman, gak bakal bikin web down.

3. Cara Install + Setup Aman 5 Menit

1. Download: portswigger.net/burp → Pilih "Community Edition"
2. Buka Burp → Tab Proxy → Options → Proxy Listener ON port 8080
3. Setting Browser → Proxy → Manual → IP 127.0.0.1 Port 8080
4. Buka browser ke web DVWA kamu → Semua request bakal ketangkep di Burp
5. Selesai test → Matikan proxy browser biar internet normal lagi

4. Workflow Hacker White Hat Pakai Burp

Step 1: Nyalain Proxy → Buka DVWA login page
Step 2: Isi username/password salah → Klik Login
Step 3: Liat di Burp Tab "HTTP history" → Klik request login tadi
Step 4: Liat Tab "Params" → Di sinilah parameter username & password
Step 5: Send to Repeater → Ganti password → Send → Liat response
Goal: Paham alur data, bukan ngehack
🔥 Misi Pertama Kamu Pakai Burp:
1. Install Burp Community + DVWA di laptop
2. Tangkap request login DVWA pakai Proxy
3. Kirim ke Repeater → Ganti username jadi "admin"
4. Kirim → Liat response "Login failed". Itu normal & aman.
Skill ini = dasar semua pentester profesional
