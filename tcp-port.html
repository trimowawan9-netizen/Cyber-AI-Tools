2.2 TCP/IP & Port

3-way handshake, Port 80, 443, 22, 21. Jalan tol data di internet

1. TCP 3-Way Handshake = Salaman

Sebelum ngobrol, komputer salaman dulu 3x biar koneksi aman:

1. Client → Server: SYN "Bro boleh kenalan?"
2. Server → Client: SYN-ACK "Boleh, kamu siapa?"
3. Client → Server: ACK "Gua si A, gas ngobrol"
Baru data dikirim!

2. Port = Nomor Pintu Rumah

1 IP bisa punya 65535 port. Tiap port buat aplikasi beda:

Port Layanan Catatan Hacker
80 HTTP Web biasa, gak dienkripsi
443 HTTPS Web aman, pake SSL/TLS
22 SSH Remote server Linux. Target empuk
21 FTP Transfer file, password polos

3. Command Wajib Hacker

netstat -an → Liat port yang lagi dibuka di PC kamu
telnet google.com 80 → Cek port 80 nyambung gak
nmap -p 22,80,443 target.com → Scan 3 port penting
🔥 Latihan 1 Menit:
1. Buka CMD/Terminal ketik: netstat -an
2. Cari tulisan LISTENING → itu port yang lagi jaga pintu PC kamu
3. Coba telnet google
