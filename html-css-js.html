3.1 HTML CSS JavaScript Dasar

Kerangka, baju, otak website. 3 serangkai yang harus dikuasai hacker web

1. HTML = Kerangka Rumah

HTML = Tulang + Tembok. Bikin struktur web: judul, paragraf, tombol, form

<h1>Judul Website</h1>
<p>Ini paragraf</p>
<input type="text" name="username">
<button>Login</button>

2. CSS = Baju + Cat Tembok

CSS = Desain. Bikin web cantik: warna, ukuran, posisi

h1 { color: red; font-size: 30px; }
button { background: #00ff88; border-radius: 10px; }
.login-box { width: 300px; margin: auto; }

3. JavaScript = Otak + Otot

JS = Yang bikin web gerak. Klik tombol muncul popup, validasi form, ganti isi web

<script>
alert('Hacked!'); // Muncul popup
document.getElementById('username').value = 'admin';
</script>

4. Kenapa Hacker Wajib Ngerti Ini?

XSS Attack: Suntik JS ke HTML korban
SQLi: Manfaatin form input HTML
DOM: JS ngubah HTML di browser korban
🔥 Latihan 1 Menit:
1. Klik kanan web → Inspect → tab Elements
2. Klik tag <h1> ganti tulisannya jadi "Saya Hacker"
3. Tab Console ketik: alert('Lu kena XSS') lalu Enter
4. Itu konsep XSS dasar bro! Cuma di browser kamu doang
