2.1 IP Address & Subnet

IPv4, IPv6, CIDR, Public vs Private IP. Alamat rumah komputer di internet

1. IP Address = Alamat Rumah

Setiap device nyambung internet punya alamat unik. Biar data gak nyasar.

IPv4: 192.168.1.1 → 4 angka, 0-255
IPv6: 2001:0db8:85a3:: → 8 blok, buat gantiin IPv4 yang abis
Public IP: Alamat internet kamu, keliatan dari luar
Private IP: Alamat lokal, cuma di wifi rumah: 192.168.x, 10.x

2. Subnet & CIDR = Kode Pos

CIDR itu cara nulis "blok IP" biar gak nulis satu-satu. /24 artinya 255.255.255.0

192.168.1.0/24 = IP dari 192.168.1.1 sampe 192.168.1.254
10.0.0.0/8 = Blok gede banget, buat perusahaan
/32 = 1 IP doang, /0 = semua IP di dunia

3. Cara Baca IP Hacker

8.8.8.8 = DNS Google
127.0.0.1 = Localhost = komputer kamu sendiri
0.0.0.0 = Semua interface, buat dengerin koneksi masuk
🔥 Latihan 1 Menit:
1. Buka CMD/Terminal ketik: ipconfig Windows / ifconfig Linux
2. Cari "IPv4 Address" itu IP private kamu di wifi
3. Google "what is my ip" → itu IP public kamu
