1.1 OS Basics

Kenalan sama Windows, Linux, macOS. File system, permission, proses

1. Apa itu Operating System?

OS = Otaknya komputer. Ngatur hardware + kasih jalan ke aplikasi.

3 OS Wajib Hacker:
1. Linux - Raja server + tools hacker. Kali Linux = Linux + tools pentest
2. Windows - Target paling banyak, soalnya user paling banyak
3. macOS - Basis Unix, command-nya mirip Linux

2. File System

Lemari penyimpanan OS. Beda OS beda struktur:

Linux/macOS: /home/bro/dokumen
Windows: C:\Users\Bro\Documents

3. Permission - Kunci File

Siapa boleh baca/tulis/eksekusi. Di Linux pake angka: chmod 777 file.txt

7 = baca+tulis+eksekusi bebas
4 = cuma baca
0 = gak boleh apa-apa

4. Proses - Program Yang Jalan

Program yang lagi aktif = proses. Cek pake:

Linux: ps aux atau top
Windows: Ctrl+Shift+Esc → Task Manager
🔥 Latihan 30 detik: Buka terminal/CMD, ketik whoami Enter. Itu nama user kamu di OS.
